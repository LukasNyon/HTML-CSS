principais formatações

negrito / destaque

Nessa frase temos temos em negrito usando a tag B( não semantica )

porem, Nessa frase temos uma frase em destaque usando o termo STRONG. ( semântica ).

Itálico / Ênfase

agora temos um termo em italico,utlizando a tag I ( nao sêmantica )

também temos outra forma de da ênfase as frase, utilizando a tag EM ( sêmantica. )

Texto marcado

podemos criar tambêm um texto marcado usando a TAG mark (semantica).

tambêm podemos mudar a cor da marcação abrindo um parâmentro style ( não semantica ).

Texto grande e pequeno

estamos criando um texto grande, e um texto pequeno nesse parágrafo

texto deletado

podemos marca um texto que foi excluído para indicar que ele deve ser lido, mas não considerado

texto inserido

podemos marca um texto como inserido, para da enfâse e indicar que ele foi adicionado depois

Texto Sobrescrio

frases que normalmente tem palavras como x^20.y^3, colocando a tag x20.y3

Texto Subscrito

frases que normalmente tem palavras como H2O, colocando a tag H2O
